安装

  1. 配置 application/config/config.php 为你的站点url，配置 application/config/site_info.php 中的站点名称、appkey、secret还有最重要的taobaoke pid。
  2. 首先自己在数据库中创建一个数据库，然后配置 application/database 中的 username，password，database
  3. 访问 站点url/index.php/login/install ，输入管理员的email和密码
  4. 访问 站点url/index.php/login 登录
  5. 可选 修改 application/views/home.php 底部的统计代码为你自己的百度统计或者Google Analytics.

警告 请严格按照上述顺序进行，不要跳过某一步骤，否则可能出现错误。

 

使用

  1. 新增商品类别：访问 站点url/index.php/admin/cat 新增你的站点的商品类别（类别会出现在首页和后台搜索结果页），然后修改类别slug为英文（中文url目前有bug，而且不优雅）
  2. 新增条目：访问 站点url/index.php/admin，选择类别之后搜索关键词，点击某个条目之后再选择图片，条目就会出现在首页（请选择类别之后再搜索关键词，这样条目会自动添加到该类别）
 

优化url

如果使用Apache服务器，可以配置 .htaccess 来去掉url中的index.php。的RewriteBase为程序所在文件夹目录，比如http://localhost/33pu/ 即为 /33pu/ ，比如：（如果使用**Nginx服务器**，配置见最后说明）

RewriteEngine on
RewriteBase /33pu/
# Hide the application and system directories by redirecting the request to index.php
RewriteRule ^(application|system|\.svn) index.php/$1 [L]
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ index.php/$1 [QSA,L]

如果是在根目录比如 http://localhost/ ，那么RewriteBase即为 /。

如果使用nginx服务器，对应的配置方法为：

location / {
  index index.php;
  if (-f $request_filename/index.php){
          rewrite (.*) $1/index.php;
  }
  if (!-f $request_filename){
          rewrite (.*) /index.php;
  }
}
